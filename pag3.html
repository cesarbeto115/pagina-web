Glosario de Comandos SQL



Lenguaje de Definición de Datos (DDL)


CREATE

Descripción: Crea una base de datos o tabla.

Ejemplo: CREATE TABLE empleados (id INT, nombre VARCHAR(50));

ALTER

Descripción: Modifica la estructura de una tabla existente.

Ejemplo: ALTER TABLE empleados ADD sueldo DECIMAL(10,2);

DROP

Descripción: Elimina una tabla o base de datos.

Ejemplo: DROP TABLE empleados;

TRUNCATE

Descripción: Elimina todos los registros de una tabla sin borrar su estructura.

Ejemplo: TRUNCATE TABLE empleados;

RENAME

Descripción: Cambia el nombre de una tabla.

Ejemplo: ALTER TABLE empleados RENAME TO trabajadores;



Lenguaje de Manipulación de Datos (DML)


SELECT

Descripción: Consulta datos de una tabla.

Ejemplo: SELECT * FROM empleados;

INSERT

Descripción: Inserta nuevos registros en una tabla.

Ejemplo: INSERT INTO empleados (id, nombre) VALUES (1, 'Juan');

UPDATE

Descripción: Modifica registros existentes.

Ejemplo: UPDATE empleados SET nombre = 'Pedro' WHERE id = 1;

DELETE

Descripción: Elimina registros de una tabla.

Ejemplo: DELETE FROM empleados WHERE id = 1;



Lenguaje de Control de Datos (DCL)


GRANT

Descripción: Otorga permisos a usuarios.

Ejemplo: GRANT SELECT ON empleados TO usuario1;

REVOKE

Descripción: Revoca permisos otorgados.

Ejemplo: REVOKE SELECT ON empleados FROM usuario1;



Lenguaje de Control de Transacciones (TCL)


COMMIT

Descripción: Guarda los cambios realizados.

Ejemplo: COMMIT;

ROLLBACK

Descripción: Revierte los cambios no confirmados.

Ejemplo: ROLLBACK;

SAVEPOINT

Descripción: Crea un punto de restauración.

Ejemplo: SAVEPOINT punto1;

SET TRANSACTION

Descripción: Define características de la transacción.

Ejemplo: SET TRANSACTION READ ONLY;
